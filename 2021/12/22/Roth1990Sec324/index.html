Stable matchings can be represented as extreme points of the solutions of a system of linear inequalities

Posted on 2021-12-22 Edited on 2021-12-24 In Summary Views: Disqus:

This post serves as a summary of Section 3.2.4 in the book Two-Sided Matching by Roth and Sotomayor(1990). From the last post, we know that the set of stable matchings can be thought as set of fixed points for some functions on a complete lattice, in this post, we would show a similar and as surprising result, that the set of stable matchings can be characterized as the set of extreme points of a system of linear constraints.

An extreme points of a convex set \(C\) is a point that doesn't line in any line segment entirely contained in the set \(C\). That is, for \(x \in C\) being an extreme point \(\nexists y,z \in C\) s.t. \(x=\alpha y + (1- \alpha)z\) for some \(\alpha \in (0,1)\), or equivalently, \(x=\alpha y + (1- \alpha)z\) \(\Rightarrow \alpha = 0 \text{ or } 1\).

In a market of \(M \times W\), we can represent fractional matchings by a \(|M| \times |W|\) matrix \(\mathbf{x}\), which satisfies the following constraints:
\(\sum_{j}x_{mj} \leq 1\) for all m in M (1);
\(\sum_{i}x_{jw} \leq 1\) for all w in W (2);
\(x_{mw} \geq 0\) for all m in M and all w in W (3);
One way is to interpret \(\mathbf{x}\) as lotteries and to consider \(x_{mw}\) as the probability that m is matched to w. The other way is to interpret \(\mathbf{x}\) as time-sharing arrangements and to consider \(x_{mw}\) as the time that m is matched with w.

A matrix \(\mathbf{x}\) is a (configuration of) matching if there is matching \(\mu\) s.t. \(x_{mw}=1\) if.f. \(\mu(m)=w\). We can see that every integer solutions of the above constraints (1,2,3) is a matching, where \(\sum_{j}x_{mj} = 0\) means m is unmatched and \(\sum_{i}x_{jw} = 0\) means w is unmatched.

For a matching to be individually rational, we need to impose the constraint to make sure no one is matched with unacceptable partner. Denoting \(A \equiv \{(m,w)|w \succ_m w \& m \succ_w w\}\), we can express the constraints as:
\(x_{mw}=0\) for \((m,w) \in M \times W \setminus A\) (4);

For stability, we need one more constraint:
\(\underset{j \succ_m w}{\sum}x_{mj} + \underset{i \succ_w m}{\sum}x_{iw} + x_{mw} \geq 1\) for all \((m,w) \in A\) (5)1;

It is not difficult to see every integer solution of the constraints (1-5) is an extreme points. Because if not, we can find \(\mathbf{y} \neq \mathbf{z}\) s.t. \(0<\alpha y_{ij} + (1- \alpha) z_{ij}<1\) for some \((i,j)\) violating that every element of \(\mathbf{x}\) is an integer.

To show that every extreme points of the set of solutions for the constraints (1-5) is an integer solution, we need to construct an auxiliary matrix \(\mathbf{z(x)}=\mathbf{\bar{z}(x)}-\mathbf{\underline{z}(x)}\), where
\(\bar{z}(x)_{mw}=1\) if \(w=\bar{x}(m)\) and \(\bar{z}(x)_{mw}=0\) otherwise;
\(\underline{z}(x)_{mw}=1\) if \(w=\underline{x}(m)\) and \(\underline{z}(x)_{mw}=0\) otherwise;
\(\bar{x}(m)\equiv \underset{w.r.t \succ_m}{\max}\{w \in W| x_{mw}>0\}\) is man's most preferred woman among \(\{w \in W s.t. x_{mw}>0\}\);
\(\underline{x}(m)\equiv \underset{w.r.t \succ_m}{\min}\{w \in W| x_{mw}>0\}\) is man's least preferred woman among \(\{w \in W s.t. x_{mw}>0\}\).

such that for any extreme solution \(\mathbf{x}\), we can find small enough \(\delta\), s.t. both \(\mathbf{x}-\delta \mathbf{z(x)}\) and \(\mathbf{x}+\delta \mathbf{z(x)}\) would solve (1-5), while \(\mathbf{x} = \frac{1}{2}(\mathbf{x}+\delta \mathbf{z(x)}) + \frac{1}{2} (\mathbf{x}-\delta \mathbf{z(x)})\), hence, we must require \(\mathbf{z(x)}=0\). This would imply \(\bar{x}(m)=\underline{x}(m)\) for all m, and we know \(\mathbf{x}\) is an integer solution.

For details of the proof of the statement underline above, one can refer to the book or the paper by Vande Vate (1992).

Reference

Roth, A., & Sotomayor, M. (1990). The structure of the set of stable matchings. In Two-Sided Matching: A Study in Game-Theoretic Modeling and Analysis (Econometric Society Monographs, pp. 54-77). Cambridge: Cambridge University Press. doi:10.1017/CCOL052139015X.003

Rothblum, U.G. (1992). Characterization of stable matchings as extreme points of a polytope. Mathematical Programming, 54, 57-67.

  1. We require it to be hold for all \((m,w) \in A\) instead of all \((m,w) \in M \times W\) because we allow the case of being unmacthed, e.g. when m is not acceptable to any woman and w is not acceptable to any man, we can have \(\underset{j \succ_m w}{\sum}x_{mj} = \underset{i \succ_w m}{\sum}x_{iw} = x_{mw}=0\) \(\Rightarrow \underset{j \succ_m w}{\sum}x_{mj} + \underset{i \succ_w m}{\sum}x_{iw} + x_{mw} = 0\)↩
